# 分镜脚本模板库

> 从300条脚本分镜模板中提炼的常用结构

---

## 一、经典叙事结构模板

### 1. 三幕式（适用于剧情短片/漫剧）

```
第一幕·建置（占25%时长）：介绍角色、场景、日常状态 → 触发事件
第二幕·对抗（占50%时长）：冲突升级、障碍出现、角色挣扎
第三幕·解决（占25%时长）：高潮对决、转折揭示、结局收尾
```

### 2. 情感爆发型（适用于情感漫剧）

```
铺垫（30%）：日常温馨/平淡 → 建立情感基础
矛盾（30%）：冲突/误会/分离 → 情绪积累
爆发（20%）：情绪顶峰 → 哭戏/对峙/告白
余韵（20%）：释然/遗憾/希望 → 画面定格+旁白
```

### 3. 动作对决型（适用于武侠/战斗漫剧）

```
对峙（20%）：双方亮相、气氛紧张、武器展示
交锋（40%）：攻防转换、技能展示、环境破坏
高潮（20%）：致命一击/绝招释放
结局（20%）：胜负分晓、战后姿态
```

## 二、分镜节奏模板

### 快节奏（紧张/动作/悬疑）
- 每镜头 1-2 秒
- 频繁景别切换（特写↔中景↔远景）
- 快速运镜（急推/急拉/手持晃动）
- 硬切转场

### 慢节奏（情感/唯美/叙事）
- 每镜头 3-5 秒
- 缓慢景别过渡（远景→中景→近景→特写）
- 平稳运镜（缓慢推近/横移/升降）
- 渐变/叠化转场

### 混合节奏（张弛有度）
- 铺垫段慢节奏 → 冲突段快节奏 → 高潮段极快 → 结尾段慢节奏

## 四、转场方式参考

| 转场 | 效果 | 适用场景 |
|------|------|----------|
| 硬切 | 直接切换，节奏快 | 动作戏、紧张场景 |
| 叠化 | 前后画面交叉淡入淡出 | 时间流逝、回忆闪回 |
| 黑场 | 先黑屏再切入新场景 | 场景大幅转换、情绪转折 |
| 匹配剪辑 | 前后镜头有形状/动作相似 | 流畅过渡、视觉连贯 |
| 动作转场 | 利用角色动作衔接 | 追逐、连续动作 |
| 遮挡转场 | 画面被物体遮挡后切换 | 走过柱子、经过门洞 |
| 闪白 | 白光闪烁后切换 | 回忆、震撼、爆炸 |

## 五、声音设计模板

### 配乐风格对照

| 情绪 | 乐器/风格 | 节奏 |
|------|-----------|------|
| 史诗壮阔 | 交响乐团+大鼓 | 渐强→高潮 |
| 悲伤凄美 | 二胡+箫+古筝 | 缓慢、留白 |
| 紧张悬疑 | 弦乐颤音+低频鼓 | 持续推进 |
| 温馨治愈 | 钢琴+吉他+轻弦乐 | 轻柔、平稳 |
| 热血激昂 | 电吉他+快鼓点+管乐 | 快节奏、爆发 |
| 古风雅致 | 古琴+编钟+笛子 | 悠扬、从容 |

### 音效分类

| 类型 | 示例 |
|------|------|
| 环境音 | 风声、雨声、鸟鸣、虫鸣、海浪、城市喧嚣 |
| 动作音 | 脚步声、衣物摩擦、金属碰撞、弓弦拉满 |
| 情绪音 | 心跳加速、呼吸加重、沉默中的钟摆 |
| 特殊音 | 魔法释放、能量聚集、时空扭曲、爆炸 |
