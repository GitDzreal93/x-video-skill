# AI漫剧提示词公式与模板

## 一、文生图提示词公式

### 核心公式

```
[风格] + [场景/背景] + [主体描述+行为] + [环境细节] + [光影效果] + [构图方式] + [景别]
```

### 公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 风格 | 整体美术风格定位 | 国漫3d风格 / 电影级写实 / 水墨武侠 / 赛博朋克 |
| 场景/背景 | 画面发生的环境 | 古风宫斗场景，皇后宫中 / 荒野戈壁，烈日当空 |
| 主体+行为 | 核心人物或物体的动作状态 | 皇后跪坐眼帘低垂 / 后羿拉弓瞄准天空 |
| 环境细节 | 补充氛围的次要元素 | 背景是模糊宫墙 / 落叶纷飞，花瓣飘零 |
| 光影效果 | 光线与阴影的描述 | 淡黄光影 / 逆光剪影 / 丁达尔效应光线穿透 |
| 构图方式 | 画面布局方式 | 构图居中 / 构图对称 / 构图偏左 |
| 景别 | 镜头远近 | 面部特写 / 近景 / 中景 / 全景 / 远景 |

### 提示词范例

**古风宫斗场景**
```
国漫3d风格，古风宫斗场景，皇后宫中，皇后跪坐眼帘低垂，泪珠滑落脸颊，背景是模糊宫墙，淡黄光影，写实细腻，构图居中，面部特写
```

**水墨武侠场景**
```
中国水墨武侠风格，枫叶林中，两位侠客对峙手持兵器，枫叶飘落在空中，晨雾弥漫，侧逆光，意境深远，构图对称，远景
```

**赛博朋克场景**
```
赛博朋克风格，未来都市霓虹街道，身穿机械装甲的角色行走在雨中，霓虹灯倒映在积水路面，紫蓝色冷调灯光，高对比度，构图偏右，中景
```

## 二、图生视频提示词公式

### 核心公式

```
[动作描述] + [镜头运动] + [情绪/照明]
```

### 公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 动作描述 | 画面中主体的动态变化 | 皇后摇头呢喃 / 角色拔剑前冲 |
| 镜头运动 | 虚拟摄像机的移动方式 | 镜头微晃 / 镜头急推 / 镜头拉远 |
| 情绪/照明 | 情感氛围与光线变化 | 崩溃情绪，淡黄柔光照明 / 紧张氛围，冷白光闪烁 |

### 提示词范例

**情感戏**
```
皇后摇头呢喃，镜头微晃，崩溃情绪，淡黄柔光照明
```

**动作戏**
```
角色拔剑冲刺，镜头快速跟随，紧张激烈，冷白侧光
```

**风景戏**
```
镜头从山顶缓缓升起，云海翻涌，壮阔氛围，自然光渐变
```

## 三、角色三视图提示词

### 什么是三视图？

角色三视图（Turnaround / Character Sheet）是 AI 漫剧制作中保证角色跨镜头一致性的核心参考图。在生成分镜画面之前，先用三视图锁定角色的外观。

### 三视图提示词公式

```
[风格] + [角色全身描述] + [角色三视图/多角度展示] + [白色/纯色背景] + [正面+侧面+背面]
```

### 三视图公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 风格 | 与全片一致的风格前缀 | 国漫3d风格 / 赛博朋克风格 / 日漫二次元风格 |
| 角色全身描述 | 详细的从头到脚外观 | 圆脸微胖青年，短发，戴黑框眼镜，穿旧格子衫和白色工作服，深色休闲裤，白色运动鞋 |
| 三视图关键词 | 触发AI生成多角度的关键词 | 角色三视图 / character turnaround / 多角度展示 / 正面侧面背面 |
| 背景要求 | 纯色背景突出角色 | 白色背景 / 纯色背景 / 无背景 |
| 角度说明 | 明确要求哪些角度 | 正面+左侧面+背面 / 前后左右四视角 |

### 三视图提示词范例

**古风角色（甄嬛皇后）**
```
国漫3d风格，角色三视图，古代皇后，梳高髻插凤钗，面部精致妆容，身着暗红色凤袍绣金丝牡丹纹，腰系玉带，足蹬绣花凤头鞋，表情端庄威严，白色纯净背景，正面+左侧面+背面，写实细腻
```

**赛博朋克角色（阿程）**
```
国漫3d风格，角色三视图，圆脸微胖青年约30岁，短寸头发，戴黑框眼镜，穿旧格子衫外搭白色SPA工作服，深灰色休闲裤，白色旧运动鞋，手腕上有旧款电子手表，表情疲惫但眼神温和，白色纯净背景，正面+左侧面+背面，写实细腻
```

**机械仿生角色（小七）**
```
国漫3d风格，角色三视图，精致银白色仿生机器人女性，面部半机械半仿真皮肤，金属关节可见，身穿浅蓝色时尚连衣长裙，银白色长发，眼睛发出淡蓝色微光，体态修长优雅，白色纯净背景，正面+左侧面+背面，写实细腻
```

### 三视图生成要点
- **必须为每个有台词的主要角色生成三视图**
- 三视图生成后，后续所有文生图提示词都引用 `@角色三视图 作为角色形象参考`
- 配角/群演不需要三视图，用简短描述即可
- 三视图背景必须是纯色，方便后续抠图或参考

## 四、镜头首帧/尾帧提示词

### 什么是首帧和尾帧？

- **首帧**：镜头开始时的画面定格，是图生视频的起始参考图
- **尾帧**：镜头结束时的画面定格，与下一个镜头的首帧形成视觉衔接

首帧和尾帧的配合使用能极大提升：
1. **镜头间连贯性**：前一镜头尾帧 → 后一镜头首帧，动作和位置自然衔接
2. **图生视频精度**：AI 视频工具（Seedance等）支持首帧/尾帧输入，生成过渡更精准
3. **节奏控制**：首帧决定入场状态，尾帧决定定格姿态

### 首帧提示词公式

```
[风格] + [场景] + [角色在此镜头的起始状态/姿势] + [光影] + [构图] + [景别]
```

### 尾帧提示词公式

```
[风格] + [场景] + [角色在此镜头的结束状态/姿势] + [光影] + [构图] + [景别]
```

### 首帧/尾帧设计原则

1. **动作衔接**：前一镜头尾帧的动作 = 后一镜头首帧的动作起点
2. **位置连贯**：角色在画面中的位置前后一致
3. **情绪递进**：首帧的情绪 → 经过视频动作 → 尾帧的情绪有变化
4. **共享元素**：相邻镜头的首尾帧保留相同的背景/光影/道具

### 首帧/尾帧范例

**镜头5（按摩动作）**
```
首帧：国漫3d风格，SPA馆按摩区，青年站在按摩床旁双手抬起准备按压，暖黄灯光聚焦手部，构图偏右，近景

尾帧：国漫3d风格，SPA馆按摩区，青年双手停在银白色机械背部手指不动，面部表情微变，暖黄灯光半明半暗，构图居中，近景
```

**镜头6（发现异常）→ 镜头7（震惊）的衔接**
```
镜头6 尾帧：国漫3d风格，SPA馆按摩区，青年面部瞳孔微缩眉头微皱，手指停住，暖黄侧光，构图居中，特写
镜头7 首帧：国漫3d风格，SPA馆按摩区，青年面部嘴巴微张开始震惊，手指仍停留在背部，暖黄正面光，构图居中，面部特写
```

## 四、特殊场景提示词模板

### 1. 视频延长
```
将@视频1延长X秒（生成长度选择X秒）

延续前视频的风格和主体：
0-X秒：[新内容描述]，与前视频无缝衔接

【要求】保持角色一致性，动作连贯流畅
```

### 2. 剧情编辑/改剧情
```
基于@视频1进行编辑：

【保留】原视频的运镜/部分动作/场景
【修改】[具体修改点1]
【修改】[具体修改点2]
【颠覆】[剧情反转描述]

【要求】保持镜头连贯，只在指定位置修改
```

### 3. 角色一致性技巧
- 文生图阶段：使用固定风格前缀 + 详细角色描述（服饰、发型、面部特征）
- 图生视频阶段：引用参考图 `@图片1 作为角色形象参考`
- 跨镜头保持：色调统一 + 角色标志性配色/配饰不变

### 4. 音乐卡点
```
将图片@图N...@图1依次按顺序，卡入@音频1的卡点中

【要求】画面切换与音乐节奏同步，转场自然流畅
```

## 六、可灵 3.0 专属提示词模式

### 1. 音画同步 · 多角色对话

**公式：**
```
[场景设定]，[角色A描述]（[语气情绪]，[语种/口音]）："[台词A]"，[动作描述]。[角色B描述]（[语气情绪]，[语种/口音]）："[台词B]"，[动作描述]。
```

**示例（三人以上指代）：**
> 一位女老师和三个学生站在教室里，镜头固定，女老师说："我们不说，包的，但可以说"，这时说话人切换为穿着校服的女学生，女学生举起单手握拳说："志在必得"，这时说话人切换为穿着从第一位男学生，男学生微笑着说："万无一失"，这时说话人切换为穿着从站在最右边的男学生，男学生举起双手说："稳操胜券"

### 2. 智能分镜（AI 自动规划）

**公式：**
```
[统一场景设定]，[初始景别/角度]，[角色A][动作]，说"台词"，[镜头运动/景别切换]，[角色B][动作]，说"台词"，[环境变化]，[情绪高潮]...
```

**要点：**
- 用自然语言描述完整的场景流转
- 通过"镜头推进/拉远/切换/特写"等词汇触发分镜切换
- 台词用引号标注，说话者写在前面

**示例：**
> 欧洲别墅户外露台场景，铺着蓝白格纹桌布的餐桌旁，年轻白人女性穿蓝白条纹短袖衬衫、卡其色短裤，赤脚坐着，对面是穿白色 T 恤的年轻白人男性，镜头推进，女性晃着玻璃杯里的果汁，目光望向远处的树林，说"These trees will turn yellow in a month, won't they?"，镜头特写男性低着头说，"but they'll be green again next summer."，然后女性转头，笑着看向对面的男性，说，"Are you always this optimistic? Or just about summer?"，然后男性抬起头，看着女生说，"Only about summers with you."

### 3. 自定义分镜（逐镜头控制）

**公式（英文）：**
```
shot 1, [景别+角度] of [主体], [镜头运动], [风格/氛围描述]
shot 2, [景别+角度] of [主体], [镜头运动], [风格/氛围描述]
```

**公式（中文）：**
```
镜头1-[景别+角度]，[动作/画面描述]，[镜头运动]
镜头2-[景别+角度]，[动作/画面描述]，[镜头运动]
```

**示例（英文）：**
> shot 1, profile shot of black man driving a truck, cinematic handheld
> shot 2, frontal macro shot of black man driving a truck, cinematic handheld
> shot 3, macro shot of hands on the steering wheel, cinematic handheld
> shot 4, macro shot of a weathered picture of a young black child laying on the passenger side seat, cinematic handheld

**示例（中文）：**
> 镜头1-后远景低角度跟拍，骑手向前驶去
> 镜头2-侧方低角度近景，特写摩托车车轮
> 镜头3-骑手第一人称主观视角，前方是摩托车车把和表盘
> 镜头4-正面中景迎向摩托跟拍，骑手头盔正对镜头

### 4. 长镜头叙事（3-15秒）

**公式：**
```
[开场景别+运镜]，[色调/风格]；主体[角色描述][动作]...第 N 秒，[画面变化/镜头运动]，[情绪递进]...最后 N 秒，[镜头运动]，[画面定格/收束]。
这是一个 X 秒的电影级长镜头。一镜到底，无剪辑转场。
```

**示例：**
> 超广角中远景横向跟拍开场，稳定器低位贴地运动，冷蓝夜色与银白星空形成高对比的浪漫电影色调；主体为身穿墨绿色长裙的年轻女性，在月光照亮的花园草地上全力奔跑，裙摆被风掀起形成汹涌的动态曲线...第 4 秒，镜头随着她向前加速，背景多名身着旧时代礼服的男女从左右两侧陆续闯入画面...第 8 秒，镜头逐渐拉近至中景...第 12 秒，音乐与动作达到高潮...最后 3 秒，镜头不停，继续向前推进，女子和男子冲出人群，奔向花园尽头的星空...
>
> 这是一个 15 秒的电影级长镜头。一镜到底，无剪辑转场。

### 5. 主体参考（图生视频 + 主体一致性）

**公式：**
> 上传首帧图 + 上传主体参考图（2-4 张或视频）
> Prompt: [场景延续+动作描述+镜头运动+情绪氛围]

**要点：**
- 角色类主体：上传 2-4 张参考图或录制人物视频
- 主体参考可与音色绑定
- 已绑定音色的主体，不建议在 Prompt 中额外设置音色

### 6. 多语种 & 方言/口音

**支持语种：** 中文、英文（美式/英式/印度口音）、日文、韩文、西班牙文
**支持中文方言：** 东北口音、北京口音、台湾地区口音、粤语、四川话

**公式：**
```
[角色]（[语气情绪]，[语种/口音]）："[台词内容]"
```

**示例（方言）：**
> 固定镜头，一位年轻的咖啡师低头擦拭杯子。咖啡师抬起头看向镜头，用四川话说："诶，你来了哇。今天喝点啥子嘞，冰美式还是拿铁。今天正好来了新豆子"

**示例（粤语职场混说）：**
> 高级写字楼中，男人身体后仰，带着一种疲惫的嫌弃，用粤语说："其实……我真系唔系好 buy 你呢个 logic 啰。成个 proposal 根本 align 唔到我哋个 core value。你个 flow 咁乱，点样去 convince 个 client 呀？不如你返去 re-think 下个 angle，听朝早我要见到个 final version。"

### 7. 原生文字保留/生成

**公式：**
> [场景]，镜头聚焦[含文字物体]，[文字内容描述]，[镜头环绕/推进/特写]，旁白（[语气]）："[旁白内容]"

**示例（广告）：**
> 巴黎公寓窗边场景下，午后鎏金阳光透过百叶窗洒在香水瓶上，镜头从散落的玫瑰花瓣缓缓推进，焦点移向 Kling 香水瓶的切割面，捕捉金色刻字与瓶身的光影流动，旁白（慵懒法式女声，英式口音）：Bathe in the golden hour. 镜头拉远定格完整场景，旁白：Wrap yourself in luxury with every breath.

## 七、GPT-Image-2 提示词模式

GPT-Image-2 (OpenAI) 图片生成模型，支持 300+ 案例提炼的五种提示词模式：

### 1. 经典三段式（人像/产品）

**结构：** `[光线与镜头] + [主体描述] + [细节与氛围]`

```
35mm film photography with harsh convenience store fluorescent lighting,
intimate medium shot, [主体描述], authentic film grain, [氛围],
no plastic skin, no digital over-sharpening, no watermark
```

**核心负向提示词：**
```
no plastic skin, no digital over-sharpening, no airbrushing,
no blemishes, no oily skin, no watermark, no text
```

### 2. 文学化描述（中文海报/插画）

以散文式语言从整体到局部展开，融入文化元素、色彩意象和情绪渲染。

```
极简新中式美学风格，画面以淡雅的灰白色为底，呈现出一种纸艺剪影般的立体感。
一条S形蜿蜒的裂痕状边缘将画面分割，仿佛撕开了一层纸面，露出内部色彩斑斓的东方山水景象。
裂口内，一条蜿蜒的河流自上而下贯穿整个构图...
下方题字"东方美学"以黑色楷体书写，日期与红色印章相呼应...
```

### 3. 结构化 JSON（广告创意）
