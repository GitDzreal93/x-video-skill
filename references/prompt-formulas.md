# AI漫剧提示词公式与模板

## 一、文生图提示词公式

### 核心公式

```
[风格] + [场景/背景] + [主体描述+行为] + [环境细节] + [光影效果] + [构图方式] + [景别]
```

### 公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 风格 | 整体美术风格定位 | 国漫3d风格 / 电影级写实 / 水墨武侠 / 赛博朋克 |
| 场景/背景 | 画面发生的环境 | 古风宫斗场景，皇后宫中 / 荒野戈壁，烈日当空 |
| 主体+行为 | 核心人物或物体的动作状态 | 皇后跪坐眼帘低垂 / 后羿拉弓瞄准天空 |
| 环境细节 | 补充氛围的次要元素 | 背景是模糊宫墙 / 落叶纷飞，花瓣飘零 |
| 光影效果 | 光线与阴影的描述 | 淡黄光影 / 逆光剪影 / 丁达尔效应光线穿透 |
| 构图方式 | 画面布局方式 | 构图居中 / 构图对称 / 构图偏左 |
| 景别 | 镜头远近 | 面部特写 / 近景 / 中景 / 全景 / 远景 |

### 提示词范例

**古风宫斗场景**
```
国漫3d风格，古风宫斗场景，皇后宫中，皇后跪坐眼帘低垂，泪珠滑落脸颊，背景是模糊宫墙，淡黄光影，写实细腻，构图居中，面部特写
```

**水墨武侠场景**
```
中国水墨武侠风格，枫叶林中，两位侠客对峙手持兵器，枫叶飘落在空中，晨雾弥漫，侧逆光，意境深远，构图对称，远景
```

**赛博朋克场景**
```
赛博朋克风格，未来都市霓虹街道，身穿机械装甲的角色行走在雨中，霓虹灯倒映在积水路面，紫蓝色冷调灯光，高对比度，构图偏右，中景
```

## 二、图生视频提示词公式

### 核心公式

```
[动作描述] + [镜头运动] + [情绪/照明]
```

### 公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 动作描述 | 画面中主体的动态变化 | 皇后摇头呢喃 / 角色拔剑前冲 |
| 镜头运动 | 虚拟摄像机的移动方式 | 镜头微晃 / 镜头急推 / 镜头拉远 |
| 情绪/照明 | 情感氛围与光线变化 | 崩溃情绪，淡黄柔光照明 / 紧张氛围，冷白光闪烁 |

### 提示词范例

**情感戏**
```
皇后摇头呢喃，镜头微晃，崩溃情绪，淡黄柔光照明
```

**动作戏**
```
角色拔剑冲刺，镜头快速跟随，紧张激烈，冷白侧光
```

**风景戏**
```
镜头从山顶缓缓升起，云海翻涌，壮阔氛围，自然光渐变
```

## 三、角色三视图提示词

### 什么是三视图？

角色三视图（Turnaround / Character Sheet）是 AI 漫剧制作中保证角色跨镜头一致性的核心参考图。在生成分镜画面之前，先用三视图锁定角色的外观。

### 三视图提示词公式

```
[风格] + [角色全身描述] + [角色三视图/多角度展示] + [白色/纯色背景] + [正面+侧面+背面]
```

### 三视图公式详解

| 要素 | 说明 | 示例 |
|------|------|------|
| 风格 | 与全片一致的风格前缀 | 国漫3d风格 / 赛博朋克风格 / 日漫二次元风格 |
| 角色全身描述 | 详细的从头到脚外观 | 圆脸微胖青年，短发，戴黑框眼镜，穿旧格子衫和白色工作服，深色休闲裤，白色运动鞋 |
| 三视图关键词 | 触发AI生成多角度的关键词 | 角色三视图 / character turnaround / 多角度展示 / 正面侧面背面 |
| 背景要求 | 纯色背景突出角色 | 白色背景 / 纯色背景 / 无背景 |
| 角度说明 | 明确要求哪些角度 | 正面+左侧面+背面 / 前后左右四视角 |

### 三视图提示词范例

**古风角色（甄嬛皇后）**
```
国漫3d风格，角色三视图，古代皇后，梳高髻插凤钗，面部精致妆容，身着暗红色凤袍绣金丝牡丹纹，腰系玉带，足蹬绣花凤头鞋，表情端庄威严，白色纯净背景，正面+左侧面+背面，写实细腻
```

**赛博朋克角色（阿程）**
```
国漫3d风格，角色三视图，圆脸微胖青年约30岁，短寸头发，戴黑框眼镜，穿旧格子衫外搭白色SPA工作服，深灰色休闲裤，白色旧运动鞋，手腕上有旧款电子手表，表情疲惫但眼神温和，白色纯净背景，正面+左侧面+背面，写实细腻
```

**机械仿生角色（小七）**
```
国漫3d风格，角色三视图，精致银白色仿生机器人女性，面部半机械半仿真皮肤，金属关节可见，身穿浅蓝色时尚连衣长裙，银白色长发，眼睛发出淡蓝色微光，体态修长优雅，白色纯净背景，正面+左侧面+背面，写实细腻
```

### 三视图生成要点
- **必须为每个有台词的主要角色生成三视图**
- 三视图生成后，后续所有文生图提示词都引用 `@角色三视图 作为角色形象参考`
- 配角/群演不需要三视图，用简短描述即可
- 三视图背景必须是纯色，方便后续抠图或参考

## 四、镜头首帧/尾帧提示词

### 什么是首帧和尾帧？

- **首帧**：镜头开始时的画面定格，是图生视频的起始参考图
- **尾帧**：镜头结束时的画面定格，与下一个镜头的首帧形成视觉衔接

首帧和尾帧的配合使用能极大提升：
1. **镜头间连贯性**：前一镜头尾帧 → 后一镜头首帧，动作和位置自然衔接
2. **图生视频精度**：AI 视频工具（Seedance等）支持首帧/尾帧输入，生成过渡更精准
3. **节奏控制**：首帧决定入场状态，尾帧决定定格姿态

### 首帧提示词公式

```
[风格] + [场景] + [角色在此镜头的起始状态/姿势] + [光影] + [构图] + [景别]
```

### 尾帧提示词公式

```
[风格] + [场景] + [角色在此镜头的结束状态/姿势] + [光影] + [构图] + [景别]
```

### 首帧/尾帧设计原则

1. **动作衔接**：前一镜头尾帧的动作 = 后一镜头首帧的动作起点
2. **位置连贯**：角色在画面中的位置前后一致
3. **情绪递进**：首帧的情绪 → 经过视频动作 → 尾帧的情绪有变化
4. **共享元素**：相邻镜头的首尾帧保留相同的背景/光影/道具

### 首帧/尾帧范例

**镜头5（按摩动作）**
```
首帧：国漫3d风格，SPA馆按摩区，青年站在按摩床旁双手抬起准备按压，暖黄灯光聚焦手部，构图偏右，近景

尾帧：国漫3d风格，SPA馆按摩区，青年双手停在银白色机械背部手指不动，面部表情微变，暖黄灯光半明半暗，构图居中，近景
```

**镜头6（发现异常）→ 镜头7（震惊）的衔接**
```
镜头6 尾帧：国漫3d风格，SPA馆按摩区，青年面部瞳孔微缩眉头微皱，手指停住，暖黄侧光，构图居中，特写
镜头7 首帧：国漫3d风格，SPA馆按摩区，青年面部嘴巴微张开始震惊，手指仍停留在背部，暖黄正面光，构图居中，面部特写
```

## 五、完整分镜表输出格式

### 表格列定义（完整版）

| 列名 | 说明 | 示例 |
|------|------|------|
| 拍摄场景 | 统一的场景风格描述 | 皇后宫中（国漫3D风格，古风宫斗场景，烛火摇曳，淡黄光影笼罩） |
| 镜号 | 镜头编号 | 1, 2, 3... |
| 画面内容 | 镜头中发生的事情 | 身着明黄龙袍的皇上猛拍桌案，眉头紧皱 |
| 景别 | 特写/近景/中景/全景/远景 | 特写 |
| 音乐/音效 | 声音设计 | 桌案重击声、弦乐急促起调 |
| 台词 | 角色说的话（无台词写"-"） | 皇上：你疯了？！ |
| 时长 | 单镜头时长（建议≤3s） | 2s |
| 首帧文生图提示词 | 镜头起始画面的文生图提示词 | 国漫3d风格，SPA馆按摩区，青年站在按摩床旁双手抬起... |
| 尾帧文生图提示词 | 镜头结束画面的文生图提示词 | 国漫3d风格，SPA馆按摩区，青年双手停在银白色机械背部... |
| 文生图提示词 | 镜头代表画面的文生图提示词 | 国漫3d风格，古风宫斗场景... |
| 图生视频提示词 | 用于AI生成动态视频 | 皇上猛拍桌案，镜头微颤... |

### 角色三视图输出格式（单独表/区域）

| 角色名 | 三视图文生图提示词 | 标志性特征 | 标志色 |
|--------|-------------------|-----------|--------|
| 角色A | [三视图提示词] | [便于识别的特征] | [主色调] |
| 角色B | [三视图提示词] | [便于识别的特征] | [主色调] |

## 四、特殊场景提示词模板

### 1. 视频延长
```
将@视频1延长X秒（生成长度选择X秒）

延续前视频的风格和主体：
0-X秒：[新内容描述]，与前视频无缝衔接

【要求】保持角色一致性，动作连贯流畅
```

### 2. 剧情编辑/改剧情
```
基于@视频1进行编辑：

【保留】原视频的运镜/部分动作/场景
【修改】[具体修改点1]
【修改】[具体修改点2]
【颠覆】[剧情反转描述]

【要求】保持镜头连贯，只在指定位置修改
```

### 3. 角色一致性技巧
- 文生图阶段：使用固定风格前缀 + 详细角色描述（服饰、发型、面部特征）
- 图生视频阶段：引用参考图 `@图片1 作为角色形象参考`
- 跨镜头保持：色调统一 + 角色标志性配色/配饰不变

### 4. 音乐卡点
```
将图片@图N...@图1依次按顺序，卡入@音频1的卡点中

【要求】画面切换与音乐节奏同步，转场自然流畅
```
