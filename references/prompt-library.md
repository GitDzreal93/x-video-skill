## 一、镜头类型词汇（80条精选）

### 基础景别
| 词汇 | 描述 | 效果 |
|------|------|------|
| 远景 | 从远处拍摄，人物很小 | 展示环境全貌，营造孤独/壮阔感 |
| 全景 | 人物全身可见 | 展示动作和空间关系 |
| 中景 | 人物腰部以上 | 对话、日常互动首选 |
| 近景 | 人物胸部以上 | 表情和情感交流 |
| 特写 | 面部/手部/物品细节 | 强调关键细节，情感爆发 |
| 大特写 | 眼睛/嘴唇/极小细节 | 极致强调，心理揭示 |

### 镜头运动
| 词汇 | 描述 | 效果 |
|------|------|------|
| 缓慢推近 | 镜头缓缓向主体靠近 | 强调主体，制造紧张/亲密感 |
| 急速推进 | 镜头快速冲向主体 | 冲击力，震惊，情绪爆发 |
| 缓慢拉远 | 镜头缓缓远离主体 | 展现全貌，疏离感，结尾 |
| 左摇/右摇 | 镜头水平方向转动 | 展示环境，发现新元素 |
| 横移 | 镜头水平平移 | 跟随行走，展示连续空间 |
| 跟拍 | 镜头跟随主体移动 | 临场感，紧迫感 |
| 环绕 | 镜头围绕主体360度旋转 | 全面展示主体，仪式感 |
| 升镜头 | 镜头从下往上移动 | 展示高度，仰视感 |
| 降镜头 | 镜头从上往下移动 | 展示深度，俯视感 |
| 俯冲 | 镜头从高处急速下降 | 冲击力，紧张感 |
| 手持晃动 | 镜头不规则晃动 | 纪实感，混乱，紧张 |
| 一镜到底 | 不切换镜头连续拍摄 | 沉浸感，空间连续 |
| 希区柯克变焦 | 推镜头同时拉焦距 | 背景压缩/拉伸，眩晕感 |

### 特殊镜头
| 词汇 | 描述 | 效果 |
|------|------|------|
| 鸟瞰镜头 | 从高空俯拍 | 上帝视角，宏大叙事 |
| 低角度仰拍 | 从低处向上拍 | 增强压迫感/威严感 |
| 高角度俯拍 | 从高处向下拍 | 渺小感，孤独感 |
| 主观镜头 | 以角色视角拍摄 | 代入感，沉浸体验 |
| 过肩镜头 | 从肩膀后方拍摄 | 对话场景，第三视角 |
| 反光面镜头 | 通过水面/镜面反射拍摄 | 双重空间，诗意 |
| 双重焦点 | 前景背景同时清晰 | 展示两个空间关系 |
| 时间冻结 | 人物静止背景运动 | 强调瞬间，超现实感 |
| 快闪推近 | 突然推进脸部 | 冲击力，情绪爆点 |
| 裂窗镜头 | 透过破碎玻璃拍摄 | 惊恐，破碎感 |
| 错位双画面 | 左右分屏 | 对比两种情绪/状态 |
