# AI反推提示词三步法

> 从参考图片反向推理出可用于 AI 生图的完整提示词

---

## 三步法

### Step 1: 拆解图片

要求 AI 对参考图片进行全面分析，输出结构化描述。

```
先帮我拆解这张图片，重点分析：
- 主体 (Subject)
- 场景 (Scene)
- 风格 (Style)
- 色调 (Color Tone)
- 构图 (Composition)
- 细节 (Details)

尽量具体一点，不要只做表面描述
```

### Step 2: 反推提示词

基于 Step 1 的分析结果，生成完整的正向提示词。

```
根据刚才的分析结果，帮我反推出一版完整的AI生图提示词，
要求尽量还原原图的氛围、风格和画面质感
```

### Step 3: 优化输出

将生成的提示词进一步优化为清晰、可直接复制使用的版本。

```
再把这版提示词优化成更适合直接生图的版本，
要求表达更清晰、细节更完整，可以直接复制
```

---

## 应用场景

### 场景一：参考图 → 新图
看到喜欢的图片风格，反推提示词后修改主体/场景生成新图。

### 场景二：实拍图 → AI风格化
用手机拍摄的照片作为输入，反推提示词后转为特定风格。

### 场景三：竞品分析 → 风格借鉴
分析优秀作品的画面结构，理解其提示词构成后融入自己的创作。

---

## 注意事项
- 反推提示词适用于 GPT-Image-2 / DALL-E / Midjourney 等主流图片生成模型
- 对于复杂构图，建议多轮迭代：先反推 → 生成 → 不满意再优化
