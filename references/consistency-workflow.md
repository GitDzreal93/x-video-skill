# AI视频一致性"硬控"三步法

> 锁人物 → 锁场景 → 锁衔接：AI短剧分段制作中的一致性控制实战工作流

---

## 核心原则

AI视频的一致性本质是确保多帧、多段之间**风格、场景、主体**的统一：

1. **风格一致性（含色调）**：整体视觉风格和色彩调性统一
2. **场景一致性**：场景布局、环境、光线保持一致
3. **主体一致性**：人物、动物、物体特征跨镜头延续

---

## Step 1: 锁人物 — 全程一张脸

通过生成角色**三视图**（正面/侧面/背面多角度），作为人物一致性的锚点。

### 操作路径
```
点击图片 → 九宫格 → 角色三视图生成
```

### 角色提示词要点
- 时代设定：古代/现代/科幻
- 外貌特征：五官、肤色、发型
- 服装描述：从头到脚完整装束
- 肤质细节：冷白皮/小麦色，陶瓷般/细腻
- 背景要求：干净/纯色背景

### 角色提示词示例
| 角色 | 提示词要点 |
|------|-----------|
| 女主 | 中国古代古装少女，古偶剧风格，宋代发髻，左胸像右全身，冷白皮，陶瓷般水嫩肤质，干净背景 |
| 男主 | 现代偶像剧风格，帅气，家居服，左胸像右全身，小麦色肌肤，26岁东方帅哥，干净背景 |

---

## Step 2: 锁场景 — 画面不跳戏

构建**场景资产库**，用空场景图 + 影视级场景描述建立场景参考。

### 场景资产库提示词
```
古代大户人家闺房场景，空场景图，影视级场景设计
```

### 场景一致性实现方法
1. **参照重绘法**：参考图 → 生成相似场景 → 在该场景中生成主体
2. **细节描述法**：用详细的提示词描述场景的每一个视觉要素

### 15秒片段提示词结构
每个15秒片段需包含：
- 镜头类型（推/拉/摇/移）
- 景别（近景/全景/特写）
- 人物动作 + 对话
- 最后镜头不加背景音乐（方便剪辑衔接）

---

## Step 3: 锁衔接 — 碎片拼成短片

两段15秒片段之间的无缝衔接方法：

| 方法 | 操作 | 推荐度 |
|------|------|--------|
| 方法一 | 直接延续提示词生成下一段 | ⭐⭐ |
| 方法二 | 截图尾帧 + 基于尾帧图输入下一段提示词 | ⭐⭐⭐ |

### 尾帧衔接法
1. 点击视频窗口右下角【截图尾帧】功能，得到尾帧图像
2. 基于尾帧图像输入下一段提示词，确保视觉无缝衔接

### 关键帧策略
四种关键帧类型用于保持一致性：
- **首帧** — 建立初始视觉
- **尾帧** — 收束叙事（也是下一步衔接的关键参考）
- **中间关键帧** — 推动情节
- **前后关键帧** — 强化连贯性

## 与SKILL工作流的整合

本三步法可融入 `漫剧:全流程` 的质量检查环节：
- Phase 3（分镜）→ 检查场景一致性
- Phase 4（文生图）→ 使用三视图作为人物锚点
- Phase 6（生视频）→ 尾帧衔接确保片段连贯
