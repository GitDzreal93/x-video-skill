# 美术风格设计指南

> AI漫剧的视觉设计核心：场景、色彩、光影、构图、道具、服装

---

## 一、世界观构建

### 风格定位原则
- **风格服务于主题**：根据故事类型确定视觉基调，形成统一美学语言
- **参考真实或虚构原型**：历史题材需考据时代特征；奇幻题材可融合文化符号
- **制作情绪板（Mood Board）**：收集色彩、纹理、参考图片，直观传递风格调性

### 场景设计：空间即角色
- **场景隐喻情感**：用环境暗示人物心理或剧情走向
  - 狭窄空间 → 压抑、束缚（如半地下室象征阶层边缘化）
  - 开阔场景 → 自由、希望（如山顶、大海）
  - 神秘空间 → 未知、奇幻（如旋转楼梯、烛光穹顶）
- **功能性与视觉平衡**：场景需符合镜头调度空间，同时具备视觉记忆点

## 二、色彩心理学

### 主色调情绪对照表

| 色调 | 色系 | 情绪效果 | 适用题材 |
|------|------|----------|----------|
| 冷色调 | 蓝/绿/灰 | 冷静、孤独、压抑、未来感 | 悬疑、科幻 |
| 暖色调 | 红/黄/橙 | 温暖、激烈、危险、亲密 | 爱情、战争 |
| 对比色 | 红绿/蓝橙 | 冲突、张力、戏剧性 | 对抗、高潮 |
| 低饱和 | 灰褐/暗色 | 沉闷、怀旧、压抑 | 年代戏、悲剧 |
| 高饱和 | 明亮纯色 | 活力、梦幻、夸张 | 奇幻、日漫 |
| 黑白 | 无彩色 | 严肃、经典、永恒 | 历史、艺术 |

### 色彩随剧情变化
- **递进暗示转折**：明亮→暗调表示从希望到绝望；暗调→明亮表示反转
- **角色专属色**：为核心人物设计标志性色彩，强化身份记忆
  - 例：主角穿白色（纯洁），反派穿深色（阴暗），转变时颜色渐变

### 漫剧常用配色方案

| 方案 | 主色 | 辅色 | 点缀色 | 适用 |
|------|------|------|--------|------|
| 古风宫斗 | 暗红+金黄 | 墨绿+深紫 | 金色+白色 | 宫廷权谋 |
| 水墨武侠 | 黑白灰 | 淡青+赭石 | 朱红点缀 | 武侠侠义 |
| 赛博朋克 | 深蓝+紫色 | 霓虹粉+青绿 | 霓虹黄 | 未来科幻 |
| 日漫热血 | 亮蓝+白色 | 橙色+红色 | 金色光芒 | 少年冒险 |
| 治愈日常 | 米白+浅木色 | 浅绿+淡粉 | 暖黄光 | 生活温馨 |

## 三、光影设计

### 基本光型

| 光型 | 效果 | 适用场景 |
|------|------|----------|
| 正面光 | 清晰明亮，减少阴影 | 产品展示、明亮场景 |
| 侧光 | 强调轮廓，制造立体感 | 人物刻画、戏剧场景 |
| 逆光 | 轮廓光晕，剪影效果 | 唯美、神秘、离别 |
| 顶光 | 深眼窝阴影，诡异感 | 反派、悬疑、恐怖 |
| 伦勃朗光 | 面部三角光斑，经典人像 | 高质感人物特写 |
| 丁达尔效应 | 光线穿透粒子，可见光柱 | 神圣、梦幻、森林 |
| 烛火光 | 暖黄摇曳，亲密封闭 | 古代室内、情感戏 |

### 光影叙事技巧
- **明暗对比突出焦点**：用侧光/逆光引导观众注意力
- **光影象征权力关系**：高位者强光区，低位者阴影处
- **动态光影增强张力**：晃动烛火、闪烁霓虹灯营造不安

## 四、构图原则

### 构图类型与叙事功能

| 构图 | 方法 | 叙事效果 | 示例 |
|------|------|----------|------|
| 对称构图 | 左右/上下对称 | 秩序、庄重、压抑 | 宫殿、对峙场面 |
| 三分法 | 主体在三分线交叉点 | 自然、平衡 | 常用构图 |
| 偏心构图 | 主体偏离中心 | 不安、留白想象 | 孤独、思考 |
| 框架构图 | 通过门窗/镜子拍摄 | 窥视感、隔阂 | 悬疑、心理描写 |
| 倾斜构图 | 画面倾斜 | 失控、混乱 | 紧张、疯狂 |
| 引导线构图 | 道路/走廊/河流引导视线 | 纵深、方向感 | 追逐、旅程 |
| 极简构图 | 大面积留白+小主体 | 孤独、意境 | 武侠、文艺 |

### 景别与情绪对应

| 景别 | 画面范围 | 情绪功能 |
|------|----------|----------|
| 远景 | 环境+人物（人物很小） | 展示环境、孤独、渺小 |
| 全景 | 人物全身+周围环境 | 展示动作、空间关系 |
| 中景 | 人物腰部以上 | 对话、日常互动 |
| 近景 | 人物胸部以上 | 表情、情感交流 |
| 特写 | 面部/手部/物品 | 强调细节、情感爆发 |
| 大特写 | 眼睛/嘴唇/细部 | 极度强调、心理揭示 |

## 五、道具与服装设计

### 道具设计原则
- **关键道具推动剧情**：道具可作为故事线索贯穿始终
- **道具折射时代与阶层**：材质、款式暗示角色身份
- **极简道具强化象征**：用单一物品传递复杂含义

### 服装设计原则
- **服装贴合人物弧光**：材质/款式变化反映角色成长或堕落
  - 纯洁角色：白色/浅色系 → 黑化时转为深色
  - 权力角色：华丽金饰 → 失势时朴素简衣
- **文化符号的视觉转化**：传统服饰快速建立地域或时代标签
- **角色专属配色**：主角与配角的服装色彩形成差异化

## 六、风格一致性检查清单

在生成跨镜头提示词时，确保以下要素保持一致：

- [ ] **色调统一**：所有镜头使用同一色调方案
- [ ] **光影一致**：同一场景内光线方向和色温一致
- [ ] **角色外观**：服饰、发型、配饰在各镜头中保持不变
- [ ] **风格前缀**：每个文生图提示词开头使用相同的风格描述
- [ ] **场景元素**：相邻镜头保留共享视觉元素（如同一盏灯、同一扇窗）
- [ ] **画幅比例**：全片保持相同画面比例
